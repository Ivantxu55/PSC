Este proyecto tiene como objetivo aprender a utilizar la metodología SCRUM para la gestión de proyectos en equipos. Más que las tecnologías utilizadas, que deben ser usadas adecuadamente, por supuesto, se buscará aplicar correctamente la metodología mencionada.\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysection{\texorpdfstring{Cómo construir y ejecutar el programa}{Cómo construir y ejecutar el programa}}\label{md__r_e_a_d_m_e_autotoc_md4}

\begin{DoxyEnumerate}
\item Tener instalado Java y Maven.
\item Configurar la base de datos\+:
\begin{DoxyItemize}
\item En el archivo {\ttfamily datanucleus.\+properties} debe modificar los campos {\ttfamily Connection\+URL}, {\ttfamily Connection\+User\+Name} y {\ttfamily Connection\+Password} con los valores adecuado para realizar la conexión con su base de datos.
\item En el archivo {\ttfamily config\+\_\+database.\+sql} se encuentran las consultas SQL para crear la tabla necesaria y para insertar algunos datos de prueba.
\end{DoxyItemize}
\item Haciendo uso de Maven, compilar el programa con {\ttfamily mvn compile}.
\item Ejecutar la parte del programa deseada desde Maven\+:
\begin{DoxyItemize}
\item Servidor\+: {\ttfamily mvn jetty\+:run}
\item Cliente\+: {\ttfamily mvn exec\+:java -\/Pcliente}
\end{DoxyItemize}
